# Исправление контраста карточки в превью

## Проблема
При выборе темного фона (например, черного) и светлого текста (белого), карточка "Project Overview" оставалась белой с белым текстом, что делало её нечитаемой.

## Решение
Добавлена логика автоматического определения контрастных цветов:

### Функции определения контраста

```typescript
// Определяет, является ли цвет светлым или темным

// Возвращает контрастный цвет фона для карточки
function getContrastColor(backgroundColor: string): string {
  return isLightColor(backgroundColor) ? '#FFFFFF' : '#1F2937';
}

// Возвращает контрастный цвет текста для карточки
function getContrastTextColor(backgroundColor: string): string {
  return isLightColor(backgroundColor) ? '#1F2937' : '#F9FAFB';
}
```

### Как это работает

1. **Светлый фон** (например, белый #FFFFFF):
   - Карточка: белая (#FFFFFF)
   - Текст в карточке: темный (#1F2937)

2. **Темный фон** (например, черный #000000):
   - Карточка: темно-серая (#1F2937)
   - Текст в карточке: светлый (#F9FAFB)

### Дополнительно
- Добавлена тонкая полупрозрачная рамка для лучшего визуального разделения
- Рамка использует цвет текста с прозрачностью 20%

## Результат
Теперь карточка всегда читаема независимо от выбранной цветовой схемы:
- ✅ Белый фон + черный текст → белая карточка с черным текстом
- ✅ Черный фон + белый текст → темная карточка со светлым текстом
- ✅ Любые другие комбинации автоматически адаптируются
